# Coding Challenge 1: SQL Fundamentals & Queries

## 📋 Challenge Overview

This challenge covers the fundamental SQL concepts and techniques for querying databases effectively.

---

## 🎯 Learning Objectives

After completing this challenge, you should be able to:
- ✅ Write basic SELECT statements
- ✅ Use WHERE clauses for filtering
- ✅ Apply various SQL operators
- ✅ Sort and limit results
- ✅ Work with basic database queries

## 📍 Topics Covered

### 1. SELECT Statements
- Basic SELECT syntax
- Selecting specific columns
- Selecting all columns (*)

### 2. WHERE Clause
- Comparison operators (=, <>, <, >, <=, >=)
- Logical operators (AND, OR, NOT)
- Filtering data effectively

### 3. Operators & Conditions
- IN operator
- BETWEEN operator
- LIKE operator with wildcards (%,_)
- IS NULL / IS NOT NULL

### 4. ORDER BY
- Sorting by single column
- Sorting by multiple columns
- Ascending (ASC) and Descending (DESC)

### 5. LIMIT / TOP
- Restricting result sets
- Getting top N rows

## 💡 Tips

- 🔎 Always verify your WHERE conditions
- 📊 Use ORDER BY to organize results logically
- 📌 Start simple, then add complexity
- 📝 Comment your complex queries
- ✔️ Test with known data first
