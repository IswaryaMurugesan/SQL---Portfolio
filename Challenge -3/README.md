# SQL Coding Challenge 03 - E-Commerce Database Queries

## Overview

This challenge focuses on querying an E-Commerce database using SQL SELECT statements with various filtering conditions, operators, and logical expressions on product and sales data.

## Skills Demonstrated

* SQL Querying
* Data Filtering
* DISTINCT Keyword
* Column and Table Aliases
* Comparison Operators
* Logical Operators (AND, OR, NOT)
* IN and NOT IN Operators
* BETWEEN Operator
* LIKE Pattern Matching
* NULL Handling
* Arithmetic Calculations

## Learning Objectives

After completing this challenge, I was able to:

* Write SELECT queries using DISTINCT
* Use table and column aliases for better readability
* Apply comparison operators for filtering data
* Use logical operators to combine multiple conditions
* Filter data using IN and BETWEEN
* Perform pattern matching using LIKE and wildcards
* Handle NULL values effectively
* Perform calculations using arithmetic operators

## Database Structure

### Product Table

Stores product information including product ID, product name, and price.

### Sales Table

Stores sales transaction details including sale ID, product ID, quantity, and sale amount.
