# SQL Coding Challenge 01 - Hospital Database

## Overview

This coding challenge focuses on practicing SQL Data Definition Language (DDL) commands using a Hospital Database scenario.

## Challenge Objectives

The challenge required performing the following database operations:

### Question 1: CREATE TABLE
Create a table named `Patients` with the following fields:
- PatientID
- PatientName
- Age
- Gender
- AdmissionDate

### Question 2: ALTER TABLE - Add Column
Add a new column:
- DoctorAssigned VARCHAR(50)

### Question 3: ALTER TABLE - Modify Column
Modify the PatientName column length from:
- VARCHAR(50)
to
- VARCHAR(100)

### Question 4: RENAME TABLE
Rename the table:
- Patients → Patient_Info

### Question 5: TRUNCATE vs DROP
Demonstrate the difference between:
- TRUNCATE TABLE
- DROP TABLE

## SQL Concepts Practiced

- CREATE TABLE
- ALTER TABLE
- ADD COLUMN
- MODIFY COLUMN
- RENAME TABLE
- TRUNCATE TABLE
- DROP TABLE
